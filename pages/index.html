The Pure Programming Language

Pure is a programming language that compiles to JavaScript. It is designed to be a simple, efficient, safe and modern language. 

fun main() {
  let name = "World"
  print("Hello, " + name)
}

Features

A brief tour of features that Pure supports...

  • Statically typechecked
  • Lambdas, generics, ADTs (surpowered enumerations)
  • Mutability and imperative features
  • Extensions and implementations
  • C-like structures
  • Asynchronous system
  • Easy JS interop
